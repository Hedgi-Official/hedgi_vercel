1. **Add your secrets in Replit**

   * In the “Secrets” panel, define:

     * `HEDGI_APP_SECRET` (your SMTP/API password or key)
     * `SMTP_HOST`, `SMTP_PORT`, `SMTP_USER`

2. **Prepare your database**

   * Add a `password_reset_tokens` table (or columns on your `users` table) with fields:

     * `user_id` (FK to `users`)
     * `token_hash` (SHA-256 of the token)
     * `expires_at` (timestamp)

3. **Install Node/TS dependencies**

   ```bash
   npm install express nodemailer body-parser pg # (plus your ORM or “pg”)
   npm install --save-dev @types/express @types/nodemailer
   ```

4. **Write token-utility functions**

   * `genAndStoreToken(email)`:

     1. Securely generate a random token (`crypto.randomBytes`).
     2. Hash it (`crypto.createHash("sha256")`).
     3. Upsert into `password_reset_tokens` with 1-hour expiry.
     4. Return the plain token for the reset link.
   * `validateAndConsumeToken(token)`:

     1. Hash incoming token.
     2. Look up a non-expired record.
     3. If found, delete it and return the `user_id`; otherwise fail.

5. **Create your Express server**

   ```ts
   import express from "express";
   import bodyParser from "body-parser";
   import nodemailer from "nodemailer";
   import { genAndStoreToken, validateAndConsumeToken } from "./token-utils";

   const app = express();
   app.use(bodyParser.json());
   ```

6. **Configure Nodemailer with your secrets**

   ```ts
   const transporter = nodemailer.createTransport({
     host:   process.env.SMTP_HOST,
     port:   Number(process.env.SMTP_PORT),
     auth: {
       user: process.env.SMTP_USER,
       pass: process.env.HEDGI_APP_SECRET
     }
   });
   ```

7. **Implement the “Forgot Password” endpoint**

   ```ts
   app.post("/api/forgot-password", async (req, res) => {
     const { email } = req.body;
     const token = await genAndStoreToken(email);
     const link  = `https://your-repl-url/reset?token=${token}`;
     const html  = `<p>…<a href="${link}">Change my password</a>…</p>`;

     await transporter.sendMail({
       to:      email,
       from:    "no-reply@your-domain.com",
       subject: "Reset your password",
       html
     });
     res.json({ message: "If that email exists, you’ll get a link shortly." });
   });
   ```

8. **Implement the “Reset Password” endpoint**

   ```ts
   app.post("/api/reset-password", async (req, res) => {
     const { token, password } = req.body;
     const userId = await validateAndConsumeToken(token);
     if (!userId) return res.status(400).json({ error: "Invalid or expired token." });
     // Hash & save the new password for userId...
     res.json({ message: "Password has been reset." });
   });
   ```

9. **Build your React/TS forms**

   * **ForgotPassword.tsx** → POST `{ email }` to `/api/forgot-password`
   * **ResetPassword.tsx** → read `token` from URL → POST `{ token, password }` to `/api/reset-password`

10. **Test & deploy**

    * Verify token creation, email delivery, link click, password change.
    * Ensure HTTPS, rate-limits, and cleanup of expired tokens.
